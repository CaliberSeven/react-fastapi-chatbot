# Chatbot - Democratic Campaign Comms Aide

An AI-powered strategic messaging tool designed for Democratic campaigns. This application generates professional-grade press releases and rapid-response drafts.

Built using a FastAPI backend and a React frontend, and powered by Google Gemini AI.

Hosted on [Vercel](https://react-fastapi-chatbot.vercel.app/). FastAPI Swagger documentation can be found [here](https://react-fastapi-chatbot.onrender.com/docs)
